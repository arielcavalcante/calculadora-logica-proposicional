# Elementos

## Toggle

Os botões toggle sinalizam os valores da variável:

- verdadeiro (botão verde)
- falso (botão vermelho)

## Dropdown

No dropdown podemos escolher entre uma das opções de operação lógica:

- ~ (negação)
- ∧ (conjunção)
- ∨ (disjunção)
- → (condicional)
- ↔ (bicondicional)
- ⊻ (disjunção exclusiva)
